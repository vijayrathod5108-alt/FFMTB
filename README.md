# 🎮 Elite Esports Tournament Registration Page

यह एक मॉडर्न, गेमिंग-थीम वाला सिंगल-पेज वेब एप्लिकेशन (SPA) है, जिसे ई-स्पोर्ट्स टूर्नामेंट (Esports Tournaments) के लिए स्क्वाड रजिस्ट्रेशन और पेमेंट प्रोसेस को आसान बनाने के लिए डिज़ाइन किया गया है। 

## 🚀 फीचर्स (Features)

* **UI/UX:** डार्क गेमिंग थीम के साथ रिस्पॉन्सिव डिज़ाइन जो मोबाइल और डेस्कटॉप दोनों पर बेहतरीन दिखता है।
* **मैच और प्राइज डिटेल्स ग्रिड:** रजिस्ट्रेशन की आखरी तारीख, एंट्री फीस (₹150/-), किल बोनस और रिवॉर्ड्स की पूरी जानकारी।
* **टूर्नामेंट रूल्स:** प्लेयर्स के लिए हैकिंग और एमुलेटर बैन जैसे नियमों की स्पष्ट सूची।
* **सक्योर पेमेंट गेटवे:** * स्कैन करने के लिए QR कोड इमेज प्लेसहोल्डर।
  * डायरेक्ट मोबाइल UPI पेमेंट बटन (PhonePe, Google Pay, PayTM आदि पर रीडायरेक्ट करने के लिए Dynamic UPI Deep Linking का उपयोग)।
* **ऑटोमेटेड व्हाट्सएप सबमिशन:** फॉर्म सबमिट करते ही सारी डिटेल्स (Leader Name, WhatsApp Number, Team UIDs) ऑटोमैटिक फॉर्मेट होकर सीधे एडमिन के व्हाट्सएप नंबर (`9209096713`) पर सेंड हो जाती हैं।

---

## 🛠️ तकनीकी जानकारी (Tech Stack)

* **HTML5:** स्ट्रक्चर और फॉर्म इनपुट्स के लिए।
* **CSS3:** कस्टम ग्रिड, फ्लेक्सबॉक्स, वेरिएबल्स (`:root`) और नियॉन/पर्पल गेमिंग इफेक्ट्स के लिए।
* **JavaScript (Vanilla):** क्लाइंट-साइड वैलिडेशन और `URL Encoding` के जरिए व्हाट्सएप API इंटीग्रेशन के लिए।
